# Todo List Application

## Project Description

This is a frontend Todo List Application developed using HTML, CSS, and JavaScript.

## Features

- Add Task
- Edit Task
- Delete Task
- Mark Task as Completed
- View All Tasks
- Filter Completed Tasks
- Filter Pending Tasks
- Search Tasks
- Validation for Empty Tasks

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## Folder Structure

├── index.html
├── style.css
├── script.js
└── README.md

## How to Run

1. Download the project.
2. Open the project folder.
3. Double-click `index.html`.
4. Start managing your tasks.
